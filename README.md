### 项目简介
#### 概述
“赤兔” 实时计算平台是跨越集团大数据生态体系的重要基石，具备开箱即用、一站式开发、亚秒级延时、统一SQL语言、端到端一致性、丰富的上下游支持、安全稳定等特点的企业级实时大数据计算平台。 赤兔平台提供SQL化的IDE开发环境，UI化的任务发布及任务监控体系，支持多环境隔离及环境切换，可帮助企业快速实现实时应用开发与落地，加速企业面向实时数字化的建设进程。
